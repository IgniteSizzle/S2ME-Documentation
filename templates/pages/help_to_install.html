    Подготовка и установка

    Данный раздел будет посвящён подготовке и установке IDE и библиотек на ваш компьютер. Так же нашей задачей будет настроить IDE для начала разработки вашего первого приложения или игры!

    Системные требования

    Перед подготовкой давайте обговорим минимальные системные требования для каждого нужного приложения. Так же мы посмотрим на общие системные требования вашего пк, что бы все приложения запустились!

    1. Java Development Kit 7 (JDK 7)

    Набор инструментов для создания, компиляции и выполнения Java-программ.
    Содержит компилятор (javac), виртуальную машину Java (JVM), стандартные библиотеки и отладочные утилиты.

    Минимальные системные требования:
    • ОС: Windows XP SP3 / Vista / 7, Linux, macOS 10.7+
    • Процессор: 1 ГГц
    • ОЗУ: 512 МБ (1 ГБ рекомендуется)
    • Место на диске: ~300 МБ
    • Разрешение экрана: 1024×768

    2. Sun Java Wireless Tool Kit (SJWTK)

    Среда для разработки и тестирования мобильных приложений Java ME (MIDlet’ов).
    Позволяет компилировать, запускать и отлаживать программы в эмуляторе мобильного устройства.

    Минимальные системные требования:
    • ОС: Windows XP / Vista / 7
    • Процессор: 1 ГГц
    • ОЗУ: 512 МБ
    • Место на диске: ~200 МБ
    • Разрешение экрана: 1024×768

    3. Oracle JavaME SDK 3.4

    Современная платформа для разработки Java ME-приложений.
    Объединяет функции Wireless Toolkit, добавляя улучшенные эмуляторы, отладку и интеграцию с IDE (например, NetBeans).
    Поддерживает профили CLDC, CDC и MEEP.

    Минимальные системные требования:
    • ОС: Windows 7 / 8 (32/64-бит)
    • Процессор: 1 ГГц или выше
    • ОЗУ: 1 ГБ (2 ГБ рекомендуется)
    • Место на диске: ~400 МБ
    • Разрешение экрана: 1024×768

    4. NetBeans IDE 7.3

    Интегрированная среда разработки (IDE) для Java SE, Java EE и Java ME.
    Позволяет писать, компилировать, отлаживать и визуально проектировать приложения.
    Поддерживает плагины для Java ME SDK.

    Минимальные системные требования:
    • ОС: Windows XP / Vista / 7 / 8, Linux, macOS
    • Процессор: 1 ГГц
    • ОЗУ: 512 МБ минимум (1–2 ГБ рекомендуется)
    • Место на диске: ~700 МБ
    • Разрешение экрана: 1024×768

    5. Общая минимальная конфигурация

    # Компонент Минимум
    1 ОС Windows 7 (32/64-бит)
    2 Процессор 1 ГГц (2 ядра рекомендуется)
    3 ОЗУ 1 ГБ (лучше 2 ГБ+)
    4 Место на диске ~2 ГБ свободного места
    5 Разрешение экрана 1024×768

    6. Выбор системы для разработки

    * Windows (рекомендуется)

    Все перечисленные инструменты (особенно Java ME SDK 3.4 и Wireless Toolkit) изначально разрабатывались под Windows, имеют полную совместимость, простую установку и встроенные графические эмуляторы.

    Плюсы:
    • Полная поддержка всех компонентов “из коробки”
    • Простая установка JDK, SDK и NetBeans
    • Рабочие эмуляторы Java ME
    • Хорошая совместимость со старыми версиями Java
    Минусы:
    • Более высокая нагрузка на систему
    • Требует лицензии Windows
    • Меньше гибкости в настройках окружения


    * Linux

    JDK и NetBeans прекрасно работают на Linux, но Oracle Java ME SDK 3.4 официально не поддерживается. Однако можно использовать Wine для запуска Windows-версии SDK.

    Плюсы:
    • Стабильность и высокая производительность
    • Бесплатность и открытость системы
    • Простая установка JDK и NetBeans через пакетные менеджеры
    • Возможность автоматизации через скрипты
    Минусы:
    • Java ME SDK не работает нативно
    • Эмуляторы устройств часто не запускаются под Wine
    • Требуется ручная настройка окружения
    • Зачастую требуется настройка аудио для Эмулятора
    Решение:
    Можно установить Wine (например, sudo apt install wine) и запустить Java ME SDK 3.4 в нём.
    Работает, но с ограничениями (иногда не открывается Device Manager, не отображается UI эмулятора или при взаимодействии со звуком в эмуляторах зачастую приложение экстренно закроется, либо выдаст ошибку).


    * macOS

    JDK и NetBeans работают стабильно, но Java ME SDK 3.4 не имеет версии для macOS.

    Плюсы:
    • Отличная производительность Java SE/EE
    • Удобная установка и стабильность IDE
    Минусы:
    • Нет официальной поддержки Java ME SDK
    • Эмуляторы и Wireless Toolkit не работают нативно

    Подготовка

    Для подготовки нам потребуется скачать необходимые файлы:

    • Java Development Kit 7 (JDK 7)
    • Sun Java Wireless Toolkit (SJWT) 2.5.2
    • Oracle JavaME SDK 3.4
    • NetBeans IDE 7.3


    Так же нам потребуется сама библиотека! Скачать её можно здесь. (она пригодится на моменте знакомства с библиотекой)

    Установка

    Для начала нам потребуется (для удобства) создать в корне диска C:\ с именем на латинице. В неё мы будем устанавливать все наши приложения!

    После создания мы можем устанавливать все скачанные приложения. В момент установки выбирайте созданную папку для наших приложений! (Сначала ставит Java 7, потом остальное)

    Установочники, которые потребуют Java, обычно сами находят установленную. Но вы можете выбрать свою Java, нужно будет просто указать путь к папке Java, например: C:\your_folder\jdk1.7.0_80

    Настройка

    После скачивания и установки вам потребуется запустить наш IDE. После запуска вы увидите начальную страницу:

    После запуска, в левом, вверхнем углу нажмите на Файл, после нажмите Создать проект.
    После нажатия, вы увидите окно создания проекта! Выберите в списках JavaME, во втором списке выберите Мобильное приложение:

    На следующем окне вы увидите ошибку, это нормально, так как мы не настроили платформы! На окне с ошибкой выбираем Установить пакет SDK/платформу/эмулятор...:

    Мы должны увидеть данное окно. Справа, внизу выбираем Добавить платформу:

    В первом окне мы выбираем второй пункт, нажимаем Далее >
    А во втором пункте IDE сама попытается найти платформы, но она найдёт только SJWTK. Нам нужно добавить ещё JavaME SDK, нажимаем на Поиск дополнительных папок платформы Java ME...:

    Откроется диалоговое окно с просмотром папок. Выбираем папку с JavaME SDK, которая лежит в нашей созданной папке и нажимаем open... В моём случае у меня папка лежит в корне диска C:\:

    После выбора, диалоговое окно закроется, а мы должны увидеть вторую папку в списке! Нажимаем Далее >

    После проверки он обнаружит файлы эмуляторов и нужные библиотеки для работы стандартных приложений! Для завершения настройки платформ, нажимаем Готово

    Окно добавления платформы закроется, и мы должны увидеть, что слева появились наши платформы, смело нажимаем Закрыть:

    После закрытия мы можем увидеть что ошибка пропала, а кнопка Далее > стала доступной, нажимаем:

    Далее нам нужно создать имя для нашего проекта, замените стандартное имя на ваше, и нажимайте Далее >:

    В последнем для нас окне мы можем выбрать нашу платформу (у каждой платформы свой эмулятор, если хотите простой эмулятор, то выбирайте SJWTK, если продвинутый то выбирайте JavaME SDK и в списке устройств выбирайте JavaMEPhone1) и устройство (для каждой платформы своё устройство). Так же вы можете выбрать конфигурацию и профиль устрорйства. После того как вы со всем определились, нажимайте заветную кнопку Готово:

    Готово! Теперь вы можете писать код и производить отладку! После завершения, вы увидите редактор. Что бы запустить проект, нажмите на зелёную кнопку Play. Что бы зайти в свойства проекта, нажмите файл, после кнопку Свойства (your_project_name).

    После подготовки, кстановки и настройки, мы можем теперь перейти к изучению Midlet, Alert и Canvas!
